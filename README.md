# VoidRoute

VoidRoute is a Windows gaming network diagnostics application.

## Mission

Provide real network visibility for gamers:

- Detect active game processes
- Inspect network connections
- Measure latency, jitter, and packet loss
- Analyze routes
- Diagnose network issues
- Apply safe Windows networking fixes

VoidRoute does not fake ping improvements, simulate servers, or claim routing changes without real infrastructure.
